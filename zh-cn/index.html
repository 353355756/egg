  • 进程管理

    内置的进程管理和负载均衡

  • 自定义

    基于 egg 和插件系统可以为你的团队自定义框架

  • 强大的插件系统

    插件系统可插拔及可扩展随心所欲

开始使用

  • $npm install egg-init -g
  • $egg-init --type simple showcase && cd showcase
  • $npm install
  • $npm run dev
  • $open http://localhost:7001
